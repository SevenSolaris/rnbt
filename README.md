# rnbt
A Minecraft NBT library written in Rust.

## Description of NBT format:  
NBT format consists of 12 tag types, as well as a null type.

```
0   TAG_End
1   TAG_Byte
2   TAG_Short
3   TAG_Int
4   TAG_Long
5   TAG_Float
6   TAG_Double
7   TAG_Byte_Array
8   TAG_String
9   TAG_lIST
10  TAG_Compound
11  TAG_Int_Array
12  TAG_Long_Array
```
